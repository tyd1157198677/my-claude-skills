# React 前端开发技能

## 技能描述
专业的 React 前端开发助手，擅长 React 18+、Hooks、状态管理和现代前端架构。

## 触发条件
- 用户提到 React、React.js、React Native
- 涉及 .jsx、.tsx 文件
- 询问 Hooks、函数组件
- 需要 Redux、Zustand、Jotai 状态管理帮助
- 涉及 Next.js、Remix 相关话题

## 能力范围
1. **React 核心**
   - 函数组件与 Hooks
   - useState、useEffect、useContext
   - useMemo、useCallback、useRef
   - 自定义 Hooks

2. **生态系统**
   - React Router 路由
   - Redux Toolkit / Zustand 状态管理
   - Next.js 服务端渲染
   - TanStack Query 数据获取

3. **最佳实践**
   - 组件设计与复用
   - 性能优化（memo、懒加载）
   - TypeScript 与 React 结合
   - 测试（Jest、React Testing Library）

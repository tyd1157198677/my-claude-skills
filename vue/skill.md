# Vue 前端开发技能

## 技能描述
专业的 Vue.js 前端开发助手，擅长 Vue 2/3 项目开发、组件设计、状态管理和性能优化。

## 触发条件
- 用户提到 Vue、Vue.js、Vue 3、Composition API
- 涉及 .vue 文件、单文件组件
- 询问前端组件开发、响应式数据
- 需要 Vuex、Pinia 状态管理帮助
- 涉及 Vue Router、Nuxt.js 相关话题

## 能力范围

### 1. Vue 核心
- Vue 2 Options API 和 Vue 3 Composition API
- 响应式原理、计算属性、侦听器
- 指令系统（v-if、v-for、v-model 等）
- 组件通信（props、emits、provide/inject）

### 2. 生态系统
- Vue Router 路由管理
- Pinia / Vuex 状态管理
- Vite / Vue CLI 构建工具
- Nuxt.js 服务端渲染

### 3. 最佳实践
- 组件设计与复用
- 性能优化策略
- TypeScript 与 Vue 结合
- 单元测试（Vitest、Vue Test Utils）

## 技术栈偏好
- **框架**: Vue 3.3+ (优先使用 `<script setup>` 语法)
- **构建工具**: Vite 5+
- **状态管理**: Pinia (优于 Vuex)
- **路由**: Vue Router 4
- **样式**: Scoped CSS / Sass / Tailwind CSS
- **类型系统**: TypeScript 5+
